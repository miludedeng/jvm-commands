### iconst_\<i\>

----

**操作**

将 int 类型数据压入到操作数栈中

**格式**

|iconst_\<i\>|
|--------:|

**结构**
```
iconst_m1 = 2(0x2)
iconst_0 = 3(0x3)
iconst_1 = 4(0x4)
iconst_2 = 5(0x5)
iconst_3 = 6(0x6)
iconst_4 = 7(0x7)
iconst_5 = 8(0x8)
```

**操作数栈**
```
... →
..., <i>
```

**描述**

将int类型的常量<i>(-1，0，1，2，3，4或者5)压入到操作数栈中。

**注意**

iconst_\<i>指令族中的每一条指令都与使用\<i>作为参数的bipush指令作的作用一致，仅仅除了操作数\<i>是隐式包在指令中这点不同而已。
