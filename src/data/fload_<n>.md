### fload_\<n\>

----

**操作**

从局部变量表加载一个float类型值到操作数栈中

**格式**

|fload_\<n\>|
|--------:|

**结构**
```
fload_0 = 34(0x22)
fload_1 = 35(0x23)
fload_2 = 36(0x24)
fload_3 = 37(0x25)
```

**操作数栈**
```
... →
..., value
```

**描述**

\<n\>代表一个当前栈帧中局部变量表的索引值，\<n\>作为索引定位的局部变量必须为float类型，记作value。指令执行后，value将会压入到操作数栈栈顶。

**注意**

fload_\<n\>指令族中的每一条指令都与使用\<n\>作为index参数的fload指令作的作用一致，仅仅除了操作数\<n\>是隐式包在指令中这点不同而已。
