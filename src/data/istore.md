
###

----

**操作**

将一个 int 类型数据保存到局部变量表中

**格式**

|istore|
|--------:|
|index|

**结构**
```
istore = 54(0x36)
```

**操作数栈**
```
..., value →
...
```

**描述**

index是一个无符号byte型整数，它指向当前栈帧(§2.6)局部变量表的索引值，而在操作数栈栈顶的value必须是int类型的数据，这个数据将从操作数栈出栈，然后保存到index所指向的局部变量表位置中。

**注意**

istore指令可以与wide指令联合使用，以实现使用2字节宽度的无符号整数作为索引来访问局部变量表。
