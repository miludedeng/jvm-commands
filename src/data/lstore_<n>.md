### lstore_\<n>

----

**操作**

将一个 long 类型数据保存到局部变量表中

**格式**

|lstore_\<n>|
|--------:|

**结构**
```
lstore_0 = 63(0x3f)
lstore_1 = 64(0x40)
lstore_2 = 65(0x41)
lstore_3 = 66(0x42)
```

**操作数栈**
```
..., value →
...
```

**描述**

\<n>与\<n>+1共同表示一个当前栈帧(§2.6)局部变量表的索引值，而在操作数栈栈顶的value必须是long类型的数据，这个数据将从操作数栈出栈，然后保存到\<n>及\<n>+1所指向的局部变量表位置中。

**注意**

lstore_\<n>指令族中的每一条指令都与使用\<n>作为index参数的lstore
指令作的作用一致，仅仅除了操作数\<n>是隐式包在指令中这点不同而已。
