### iload_\<n>

----

**操作**

从局部变量表加载一个 int 类型值到操作数栈中

**格式**

| iload_\<n> |
| --------:   |

**结构**
```
iload_0 = 26(0x1a)
iload_1 = 27(0x1b)
iload_2 = 28(0x1c)
iload_3 = 29(0x1d)
```

**操作数栈**
```
... →
..., value
```

**描述**

\<n>代表一个当前栈帧(§2.6)中局部变量表的索引值，\<n>作为索引定位的 局部变量必须为 int 类型，记作 value。指令执行后，value 将会压入到操作数栈栈顶

**注意**

iload_\<n>指令族中的每一条指令都与使用\<n>作为 index 参数的 iload 指
令作的作用一致，仅仅除了操作数\<n>是隐式包 在指令中这点不同而已。
