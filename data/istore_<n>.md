
### istore_\<n>

----

**操作**

将一个 int 类型数据保存到局部变量表中

**格式**

|istore_\<n>|
|--------:|

**结构**
```
istore_0 = 59(0x3b)
istore_1 = 60(0x3c)
istore_2 = 61(0x3d)
istore_3 = 62(0x3e)
```

**操作数栈**
```
..., value →
...
```

**描述**

\<n>必须是一个指向当前栈帧(§2.6)局部变量表的索引值，而在操作数栈栈顶的value必须是int类型的数据，这个数据将从操作数栈出栈，然后保存到\<n>所指向的局部变量表位置中。

**注意**

istore_\<n>指令族中的每一条指令都与使用\<n>作为index参数的istore指令作的作用一致，仅仅除了操作数\<n>是隐式包在指令中这点不同而已。
