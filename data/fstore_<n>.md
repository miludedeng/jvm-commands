### fstore_\<n\>

----

**操作**

将一个float类型数据保存到局部变量表中

**格式**

|fstore_\<n\>|
|--------:|

**结构**
```
fstore_0 = 67 (0x43)
fstore_1 = 68 (0x44)
fstore_2 = 69 (0x45)
fstore_3 = 70 (0x46)
```

**操作数栈**
```
..., value →
...
```

**描述**

\<n\>必须是一个指向当前栈帧局部变量表的索引值，而在操作数栈栈顶的value必须是float类型的数据，这个数据将从操作数栈出栈，并且经过数值集合转换后得到值value’，然后保存到\<n\>所指向的局部变量表位置中。

**注意**

fstore_\<n\>指令族中的每一条指令都与使用\<n\>作为index参数的fstore指令作的作用一致，仅仅除了操作数\<n\>是隐式包在指令中这点不同而已。
